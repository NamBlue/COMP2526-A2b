COMP 2526 Assignment A2b - Game of Life 2

Run the GameOfLife2.jar file
Simply click on the GUI to automatically advance time from within the simulation!
To stop the advancement of time, click again on the GUI.

Blue = Omnivore
Magenta = Carnivore
Yellow = Herbivore
Green = Plant
Any non solid color cell = Empty Land

The darker the color, the shorter the lifeforms lifespan is.
To create a continuous simulation of the life cycle, additional Inhabitants are created every turn using the spawn method.
